# Resource Assistant

Single-file HTML resource assistant for Transitional Services offices.
"new.html" is meant to be renamed to reflect the name of a TSO, e.g., albuquerque.html or provo.html. When the local admin receives the .html, the next job is to create local resources for each of the categories.

Open the new html file  directly in a browser to use the app. It starts with a standard set of categories but no resources; press Ctrl+Alt+A, then click the Admin tab to add, edit, delete, import, export, and manage resource packages.
